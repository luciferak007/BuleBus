# Bus Reservation System

## Description
A fully functional Bus Ticket Reservation System built with Java, JSP, and Servlets.
Allows users to search for buses, book seats, and manage reservations.

## Tech Stack
* **Frontend:** JSP, HTML, CSS
* **Backend:** Java Servlets
* **Database:** MySQL (JDBC)
* **Server:** Apache Tomcat

## How to Run
1.  Clone this repository.
2.  Import into Eclipse as "Dynamic Web Project".
3.  Configure `db.properties` with your MySQL credentials.
4.  Run on Apache Tomcat 9.
